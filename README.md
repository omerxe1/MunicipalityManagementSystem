# MunicipalityManagementSystem
Municipality Management System developed with ASP.NET MVC
# Belediye Yönetim Sistemi

ASP.NET MVC ve SQL Server kullanılarak geliştirilmiş kurumsal belediye web sitesi ve yönetim paneli projesidir.

## Özellikler

### Kullanıcı Tarafı
- Ana sayfa
- Haberler
- Duyurular
- Etkinlikler
- İletişim sayfası
- Detay sayfaları
- Responsive tasarım

### Yönetim Paneli
- Güvenli giriş sistemi
- Rol bazlı yetkilendirme
- Haber yönetimi (Ekle, Güncelle, Sil)
- Duyuru yönetimi (Ekle, Güncelle, Sil)
- Etkinlik yönetimi (Ekle, Güncelle, Sil)
- Kullanıcı yönetimi
- Dashboard ekranı

## Kullanılan Teknolojiler

- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- HTML5
- CSS3
- Bootstrap
- JavaScript

## Proje Mimarisi

- MVC Tasarım Deseni
- Katmanlı Yapı
- Veritabanı Entegrasyonu
- Kimlik Doğrulama ve Yetkilendirme
